Чтобы не использовать лини, можно присвоить разным областям на странице имена и с их помощью размещать элементы. И опять нам придется внести изменения в CSS код контейнера сетки. CSS код контейнера должен быть таким:

css grid html flex float

Flexbox-разметка начинается со знакомого свойства display. В результате применения свойства display: flex элемент превращается во flex-контейнер, а его потомки — во flex-элементы
